# 外网IP查询工具

🔍 一个简单的外网IP地址查询工具，支持IPv4和IPv6地址查询，工作日查询，并自动记录查询结果

---

## 📋 项目简介

本项目通过异步请求查询本机的外网IP地址，支持IPv4和IPv6双栈查询，同时通过 ipip.net 获取IP地址的地理位置信息，并查询当日是否为工作日，将查询结果自动记录到CSV文件中，便于后续数据分析。

### 核心功能

- ✅ 异步查询IPv4和IPv6地址
- ✅ 查询IP地址及地理位置信息（通过 ipip.net）
- ✅ 查询工作日信息（通过 iamwawa.cn）
- ✅ 飞书机器人消息通知
- ✅ 自动记录查询日志
- ✅ 将查询结果保存到CSV文件
- ✅ 使用Unix时间戳记录查询时间
- ✅ 异常时CSV保存为空，便于数据分析
- ✅ 模块化设计，职责分离清晰
- ✅ 支持请求超时设置
- ✅ 输出路径信息（工作目录、项目路径、解释器路径）

## 🔧 配置说明

### IP查询服务

| IP类型 | 查询URL | 说明 |
|--------|---------|------|
| IPv4 | `http://4.ipw.cn` | 查询外网IPv4地址 |
| IPv6 | `http://6.ipw.cn` | 查询外网IPv6地址 |
| Location | `http://myip.ipip.net` | 查询IP地址及地理位置信息 |

### 工作日查询服务

| 服务 | 查询URL | 说明 |
|------|---------|------|
| 工作日API | `https://www.iamwawa.cn/workingday/api` | 查询指定日期是否为工作日 |

### 飞书机器人配置

配置文件：`config/FeiShu.csv`

| 字段 | 说明 |
|------|------|
| `tag` | 机器人标识（用于指定发送目标） |
| `url` | 飞书机器人Webhook地址 |
| `mode` | 发送模式（none/text/post） |

**发送模式说明**：
- `none`：不发送消息
- `text`：发送纯文本消息
- `post`：发送带标题的消息

## 📊 数据格式

### 日志文件

- **位置**：`log/` 目录
- **命名格式**：`YYYYMMDD.log`（如 `20260401.log`）
- **保留策略**：默认保留30天
- **日志级别**：INFO、ERROR

### CSV数据文件

- **位置**：`db/ip_records.csv`
- **字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `timestamp` | 整数 | Unix时间戳 |
| `ip_type` | 字符串 | IP类型（IPv4/IPv6） |
| `url` | 字符串 | 查询URL |
| `ip_address` | 字符串 | IP地址（异常时为空） |

### CSV文件示例

```csv
timestamp,ip_type,url,ip_address
1774930569,IPv4,http://4.ipw.cn,***.***.***.***
1774930569,IPv6,http://6.ipw.cn,
```

## 📝 代码结构

### main.py

主入口文件，负责：
- 日志系统初始化
- 输出路径信息
- 调用IP查询功能

### main.bat

Windows批处理启动文件，负责：
- 设置编码为UTF-8
- 执行Python脚本

### ip_fetcher.py

IP查询功能模块，包含：
- `IPFetcher` 类：封装所有查询功能
  - `_fetch_url()`：通用URL请求方法
  - `fetch_all_ips()`：获取所有IP地址
  - `fetch_workingday()`：查询工作日信息
  - `fetch_all_data()`：获取所有数据（IP和工作日）
  - `get_recent_ips()`：获取最近指定秒数内的IP记录
  - `display_results()`：显示查询结果
  - `log_summary()`：输出日志摘要
  - `_save_to_csv()`：保存结果到CSV文件
  - `run()`：执行完整查询流程
- `main()` 函数：主逻辑函数

### feishu_notify.py

飞书机器人通知模块，包含：
- `FeishuNotifier` 类：管理飞书机器人通知
  - `_load_configs()`：加载飞书机器人配置
  - `_build_message()`：构建消息体（支持text/post模式）
  - `_send_to_webhook()`：发送消息到指定Webhook
  - `send_message()`：发送消息（可指定机器人tag）
  - `send_to_all()`：发送消息到所有机器人
  - `send_to_tag()`：发送消息到指定标签的机器人
- `send_feishu_message()` 函数：快捷发送飞书消息

## 🚨 错误处理

| 错误类型 | 处理方式 |
|----------|----------|
| **网络超时** | 记录异常日志，IP地址字段保存为空 |
| **DNS解析失败** | 记录异常日志，IP地址字段保存为空 |
| **连接失败** | 记录异常日志，IP地址字段保存为空 |

**异常处理逻辑**：
- 所有网络异常都会被捕获并记录
- CSV文件中IP地址字段为空表示查询失败
- 日志文件中包含详细的异常信息
