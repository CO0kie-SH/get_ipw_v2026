# 外网IP查询工具

🔍 一个简单的外网IP地址查询工具，支持IPv4和IPv6地址查询，并自动记录查询结果

---

## 📋 项目简介

本项目通过异步请求查询本机的外网IP地址，支持IPv4和IPv6双栈查询，同时通过 ipip.net 获取IP地址的地理位置信息，并将查询结果自动记录到CSV文件中，便于后续数据分析。

### 核心功能

- ✅ 异步查询IPv4和IPv6地址
- ✅ 查询IP地址及地理位置信息（通过 ipip.net）
- ✅ 自动记录查询日志
- ✅ 将查询结果保存到CSV文件
- ✅ 使用Unix时间戳记录查询时间
- ✅ 异常时CSV保存为空，便于数据分析
- ✅ 模块化设计，职责分离清晰

### IP查询服务

| IP类型 | 查询URL | 说明 |
|--------|---------|------|
| IPv4 | `http://4.ipw.cn` | 查询外网IPv4地址 |
| IPv6 | `http://6.ipw.cn` | 查询外网IPv6地址 |

## 📊 数据格式

### 日志文件

- **位置**：`log/` 目录
- **命名格式**：`YYYYMMDD.log`（如 `20260331.log`）
- **保留策略**：默认保留30天
- **日志级别**：INFO、ERROR

### CSV数据文件

- **位置**：`db/ip_records.csv`
- **字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `timestamp` | 整数 | Unix时间戳 |
| `ip_type` | 字符串 | IP类型（IPv4/IPv6） |
| `url` | 字符串 | 查询URL |
| `ip_address` | 字符串 | IP地址（异常时为空） |

### CSV文件示例

```csv
timestamp,ip_type,url,ip_address
1774930569,IPv4,http://4.ipw.cn,***.***.***.***
1774930569,IPv6,http://6.ipw.cn,
```

## 📝 代码结构

### main.py

主入口文件，负责：
- 日志系统初始化
- 调用IP查询功能

### ip_fetcher.py

IP查询功能模块，包含：
- `IPFetcher` 类：封装IP查询功能
  - `fetch_ip()`：获取单个IP地址
  - `fetch_all()`：获取所有IP地址
  - `display_results()`：显示查询结果
  - `_save_to_csv()`：保存结果到CSV文件
- `main()` 函数：主逻辑函数

## 🚨 错误处理

| 错误类型 | 处理方式 |
|----------|----------|
| **网络超时** | 记录异常日志，IP地址字段保存为空 |
| **DNS解析失败** | 记录异常日志，IP地址字段保存为空 |
| **连接失败** | 记录异常日志，IP地址字段保存为空 |

**异常处理逻辑**：
- 所有网络异常都会被捕获并记录
- CSV文件中IP地址字段为空表示查询失败
- 日志文件中包含详细的异常信息
